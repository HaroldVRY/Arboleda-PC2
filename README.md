# Caso Banco Arboleda

Repositorio base para la práctica universitaria por fases del caso Banco Arboleda.

## Cómo ejecutar
Esta entrega todavía no incluye la implementación de la app. En las siguientes fases se completarán el backend y el frontend con sus instrucciones exactas de arranque.

- Backend: `npm install` + `npm run dev`
- Frontend: `npm install` + `npm run dev`
